# PyCaesar CLI 🔐

Una herramienta interactiva de línea de comandos (CLI) construida en Python para encriptar y desencriptar mensajes de texto utilizando una versión mejorada (alfanumérica) del clásico algoritmo del Cifrado César.

## 📌 Descripción
El Cifrado César es uno de los métodos de encriptación más antiguos y conocidos. Funciona sustituyendo cada letra (y ahora número) del texto original por un carácter que se encuentra un número fijo de posiciones (saltos) más adelante.

Este proyecto fue desarrollado para aplicar conceptos fundamentales de Python, incluyendo manipulación de cadenas, funciones, condicionales, manejo de errores (Try/Except), operador módulo (%) y bucles para crear una interfaz de usuario en la terminal.

## ✨ Características (Features)
- **Soporte Alfanumérico:** Encripta tanto letras (mayúsculas y minúsculas) como números (0-9) de forma sincronizada.
- **Modo Encriptación:** Convierte texto legible en un mensaje secreto.
- **Modo Desencriptación:** Revierte el texto cifrado a su estado original usando la misma llave.
- **Validación de Entradas:** Protege el programa contra errores de usuario (ej. ingresar letras en lugar de números para la llave).
- **Interfaz Interactiva:** Menú en bucle que permite realizar múltiples operaciones sin reiniciar el programa.

## 🚀 Cómo usarlo
1. Asegúrate de tener Python instalado en tu sistema.
2. Ejecuta el archivo desde tu terminal: `python pycaesar.py`
3. Selecciona la opción deseada en el menú numérico (1 para encriptar, 2 para desencriptar, 3 para salir).
4. Escribe tu mensaje (puedes usar letras y números libremente).
5. Ingresa el "Número de saltos" (tu llave secreta, un número del 1 al 25).
6. ¡Copia tu resultado!

## 🛠️ Tecnologías utilizadas
- Python 3.x
- Lógica de desplazamiento con el operador Módulo (`%`)
- Métodos nativos de cadenas (`str.maketrans`, `translate`)
